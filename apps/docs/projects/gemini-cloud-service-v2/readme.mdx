## API Endpoints

### Universal Processing
```bash
POST /process
{
    "file_source": "file.pdf",
    "prompt": "Summarize this document",
    "features": {
        "grounding": true,
        "temperature": 0.7
    }
}
```

### YouTube Analysis
```bash
POST /youtube
{
    "url": "https://youtube.com/watch?v=...",
    "prompt": "Extract key points"
}
```

### Code Review
```bash
POST /code
{
    "code": "def hello(): print('world')",
    "language": "python",
    "prompt": "Optimize this function"
}
```

### Research Assistant
```bash
POST /research
{
    "topic": "quantum computing",
    "depth": "comprehensive"
}
```

## Supported File Types

**Documents**: PDF, TXT, RTF, DOCX, HTML, XML, JSON, CSV, MD  
**Images**: PNG, JPG, WEBP, HEIC, HEIF  
**Video**: MP4, AVI, MOV, MKV, WEBM  
**Audio**: MP3, WAV, M4A, AAC, OGG, FLAC  
**Code**: Python, JavaScript, Java, C++, Go, Rust, and more!

## Model Used

- **Gemini 2.5 Flash**: Fast, efficient, and powerful for most tasks
- **Gemini 1.5 Pro**: Enhanced multimodal capabilities for complex media

- `GEMINI_API_KEY`: Your Google AI Studio API key (required)
- `SERVICE_API_KEY`: Optional API key for service authentication
