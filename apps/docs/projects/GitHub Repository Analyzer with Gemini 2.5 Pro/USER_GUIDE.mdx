### Prerequisites

1. **GitHub Personal Access Token**
   - Generate at: https://github.com/settings/tokens
   - Required scopes: `repo` (for private repos) or `public_repo` (public only)

2. **Google AI Studio API Key**  
   - Generate at: https://aistudio.google.com/app/apikey
   - Provides access to Gemini 2.5 Pro/Flash models

### Quick Setup

```bash

cd github-repository-analyzer
python3 -m venv venv
source venv/bin/activate

# 2. Configure environment

export GOOGLE_API_KEY="your_gemini_key_here"

# 3. Verify setup

```

## üìã Basic Usage

### Analyze Your Own Repositories

```bash
# Sequential analysis (slower, backward compatible)
github-analyzer

# Concurrent analysis (recommended - 378x faster)
github-analyzer --concurrent

# High-performance with custom settings
github-analyzer --concurrent --max-concurrent 20 --memory-limit 2000
```

### Analyze Specific Users

```bash
# Analyze public repositories of any GitHub user
github-analyzer -u octocat --concurrent

# Analyze with custom output directory
github-analyzer -u microsoft -o microsoft_analysis --concurrent

# Debug mode with detailed logging
github-analyzer -u your-username --log-level DEBUG --concurrent
```

### Output Customization

```bash
# Custom output directory
github-analyzer -o my_analysis_reports --concurrent

# Different logging levels
github-analyzer --log-level WARNING  # INFO, DEBUG, ERROR
github-analyzer --log-level DEBUG    # Most verbose

# Custom environment file
github-analyzer --env-file .env.production --concurrent
```
