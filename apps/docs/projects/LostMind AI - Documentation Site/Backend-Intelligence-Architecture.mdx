# Backend Intelligence Architecture

A comprehensive backend application architecture using modern embedding models (Gemma) and RAG capabilities, designed for integration with Turborepo monorepos.

## 🚀 Features

- **RAG Pipeline** - Advanced Retrieval-Augmented Generation with conversation management
- **Excel Intelligence** - Comprehensive Excel analysis with formula intelligence and VBA analysis
- **Web Scraping Intelligence** - AI-powered web scraping with content analysis and quality assessment
- **Job Discovery Intelligence** - Intelligent job discovery with AI-powered matching and career insights
- **Modern AI Integration** - Uses latest Google GenAI SDK with Gemma embeddings
- **Scalable Architecture** - Microservices design with service orchestration
- **Production Ready** - Comprehensive API, monitoring, and deployment support

## 🛠 Quick Start

### Prerequisites

- Python 3.11+
- Google API Key for GenAI
- Vector Database (Pinecone, ChromaDB, or Weaviate)

### Installation

1. **Clone or copy the backend-architecture directory to your project**

2. **Install dependencies:**
   ```bash
   cd backend-architecture/apps/api-gateway
   pip install -r requirements.txt
   ```

3. **Configure environment variables:**
   ```bash
   cp .env.example .env
   # Edit .env with your API keys and configuration
   ```

4. **Start the API Gateway:**
   ```bash
   python main.py
   ```

The API will be available at `http://localhost:8000`

## 🔧 Configuration

All services are configured through environment variables. Key configuration options:

```bash
# Google AI Configuration
GOOGLE_API_KEY=your_google_api_key
GEMINI_MODEL=gemini-1.5-pro

# Vector Database Configuration
VECTOR_DB_PROVIDER=pinecone  # or chromadb, weaviate
PINECONE_API_KEY=your_pinecone_key
PINECONE_ENVIRONMENT=your_environment

# API Configuration
API_HOST=0.0.0.0
API_PORT=8000
```
