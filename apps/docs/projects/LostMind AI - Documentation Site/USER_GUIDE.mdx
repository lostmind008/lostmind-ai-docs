## ğŸ¨ Customizing Templates

### Template Files
- `templates/project-index.md`: Main project page template
- `templates/architecture-overview.md`: Architecture documentation
- `templates/api-reference.md`: API documentation template
- `templates/_meta.json`: Navigation structure

### Template Variables
Templates use mustache-style syntax:
- `{{projectName}}`: Project display name
- `{{projectDescription}}`: Project description
- `{{lastModified}}`: Last modification date
- `{{hasReadme}}`: Boolean for README existence
- `{{hasDocs}}`: Boolean for docs folder existence

### Creating Custom Templates
1. Create a new `.md` file in `templates/`
2. Use template variables and conditionals
3. Update extraction script to use your template

Example:
```markdown
# {{projectName}}

{{#hasCustomFeature}}
This project has custom features!
{{/hasCustomFeature}}
```
