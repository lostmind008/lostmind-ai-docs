## üîç Detailed Problem Diagnosis

### Problem Category: Installation Issues

#### Issue: Installation script fails

**Diagnosis Steps:**
1. Check script permissions: `ls -la install_lostmind_ai_framework.sh`
2. Validate script syntax: `bash -n install_lostmind_ai_framework.sh`
3. Run with verbose output: `bash -x install_lostmind_ai_framework.sh`

**Common Causes:**
- Missing Python 3 installation
- Insufficient permissions for ~/.claude directory
- Network issues during dependency installation

**Solutions:**
```bash
# Fix permissions
chmod +x install_lostmind_ai_framework.sh

# Install Python 3 (macOS)
brew install python3

# Manual directory creation
mkdir -p ~/.claude/{commands,hooks,agents}
```

#### Issue: Commands not copied during installation

**Diagnosis Steps:**
1. Check installation script logic: `grep "cp \.claude/commands" install_lostmind_ai_framework.sh`
2. Verify source files exist: `ls -la .claude/commands/ucf-*.md`
3. Check target directory permissions: `ls -la ~/.claude/commands/`

**Solutions:**
```bash
# Manual command installation
cp .claude/commands/ucf-create-task-brief.md ~/.claude/commands/
cp .claude/commands/ucf-delegate-task.md ~/.claude/commands/
cp -r .claude/hooks/* ~/.claude/hooks/
chmod +x ~/.claude/hooks/*.sh
```
