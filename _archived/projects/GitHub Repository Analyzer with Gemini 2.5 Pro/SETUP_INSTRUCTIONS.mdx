## üöÄ Quick Setup Guide

### Step 1: Install Claude Code
```bash
npm install -g @anthropic/claude-code
```

### Step 2: Set Up Global Configuration

1. **Copy CLAUDE.md to your home directory:**
```bash
cp CLAUDE.md ~/.claude/CLAUDE.md
```

2. **Set up agents:**
```bash
mkdir -p ~/.claude/agents
cp agents/* ~/.claude/agents/
```

3. **Configure MCP servers:**
   - Open `~/.claude.json` (create if doesn't exist)
   - Copy the `mcpServers` section from `mcp-config.json`
   - Replace `YOUR_MAGIC_API_KEY_HERE` with your actual API key (get one from 21st.dev)
   - Update the `filesystem` path to your home directory

### Step 3: Environment Setup

Create a `.env` file in your home directory with:
```
ANTHROPIC_API_KEY=your_claude_api_key_here
```
