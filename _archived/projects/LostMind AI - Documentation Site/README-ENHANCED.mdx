## 🚀 Installation Guide

### Prerequisites
- **Claude Code** installed and running
- **Node.js 18+** (for enhanced MCP servers)
- **Git 2.25+** (for worktree support)

### Quick Installation

#### Option 1: Automated Installation
```bash
# Download and run the enhanced installation script
curl -fsSL https://raw.githubusercontent.com/your-repo/claude-enhanced/main/install-enhanced.sh | bash
```

#### Option 2: Manual Installation

1. **Create Enhanced Directory Structure**
```bash
mkdir -p ~/.claude/{agents,commands,memory/{shared,sessions,sync},parallel,worktrees}
```

2. **Install Enhanced MCP Servers**
```bash
npm install -g \
  @modelcontextprotocol/server-sequential-thinking \
  @whenmoon/memory-mcp-server \
  @modelcontextprotocol/server-filesystem \
  @modelcontextprotocol/server-git \
  @playwright/mcp \
  @modelcontextprotocol/server-brave-search \
  @modelcontextprotocol/server-github \
  @modelcontextprotocol/server-sqlite
```

3. **Copy Enhanced Configuration Files**
```bash
# Copy all provided files to appropriate locations
cp CLAUDE-ENHANCED.md ~/.claude/CLAUDE.md
cp .claude-enhanced.json ~/.claude/.claude.json
cp agents/*.md ~/.claude/agents/
cp commands/*.md ~/.claude/commands/
```

4. **Set Permissions**
```bash
chmod 755 ~/.claude
chmod -R 644 ~/.claude/*.json ~/.claude/*.md
chmod -R 755 ~/.claude/{agents,commands,memory,parallel}
```
