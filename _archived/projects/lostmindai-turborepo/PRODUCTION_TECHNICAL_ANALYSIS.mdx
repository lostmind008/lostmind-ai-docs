### Issue #1: React 19 Dependencies (RESOLVED)

**📍 Affected Files:**
- `apps/ask/package.json`
- `apps/marketing/package.json` (likely)
- Any other React apps in workspace

**🔍 Current State:**
```json

{
  "dependencies": {
    "react": "^19.0.0",
    "react-dom": "^19.0.0"
  }
}
```

**✅ Required Fix:**
```json
// apps/ask/package.json
{
  "dependencies": {
    "react": "^19.0.0",
    "react-dom": "^19.0.0",
    "@types/react": "^19.0.0",
    "@types/react-dom": "^19.0.0"
  }
}
```

**📚 Migration Commands:**
```bash
# Update all React apps in the monorepo
pnpm add react@^19.0.0 react-dom@^19.0.0 --filter="./apps/*"
pnpm add -D @types/react@^19.0.0 @types/react-dom@^19.0.0 --filter="./apps/*"

# Verify installation
pnpm list react react-dom --filter="./apps/*"
```

**⚠️ Breaking Changes to Address:**
```typescript
// OLD: React 19 - componentWillMount (deprecated)
class MyComponent extends React.Component {
  componentWillMount() {
    // This is removed in stable React 19
  }
}

// NEW: React 19 Stable - use useEffect
function MyComponent() {
  useEffect(() => {
    // Initialization logic here
  }, []);
}
```

**📖 Reference:** [React 19 Official Release](https://react.dev/blog/2024/12/05/react-19) | [Migration Guide](https://react.dev/blog/2024/04/25/react-19-upgrade-guide)

### Issue #2: Google AI SDK Critical Incompatibility (PRODUCTION BLOCKER)

**📍 Affected Files:**
- `services/ai-compute/main.py`
- `services/ai-compute/requirements.txt`
- All AI service modules

**🔍 Current State:**
```python
# services/ai-compute/main.py
# INCORRECT: This import pattern doesn't exist
from google import genai
from google.genai import types
```

**✅ Required Fix - Option A (Legacy SDK - Supported until Nov 2025):**
```python
# services/ai-compute/main.py
import google.generativeai as genai
from google.generativeai.types import GenerationConfig

# Configure the SDK
genai.configure(api_key=settings.GEMINI_API_KEY)

# Initialize model
model = genai.GenerativeModel('gemini-2.0-flash-exp')

# Generate content
response = model.generate_content(
    prompt,
    generation_config=GenerationConfig(
        temperature=0.7,
        max_output_tokens=1000
    )
)
```

**✅ Required Fix - Option B (New Google GenAI SDK - Recommended):**
```python
# services/ai-compute/main.py
from google import genai
from google.genai import types

# Initialize client
client = genai.Client(api_key=settings.GEMINI_API_KEY)

# Generate content
response = client.models.generate_content(
    model='gemini-2.0-flash-exp',
    contents=prompt,
    config=types.GenerateContentConfig(
        temperature=0.7,
        max_output_tokens=1000
    )
)
```

**📦 Requirements Update:**
```txt
# services/ai-compute/requirements.txt
# Option A: Legacy (until Nov 2025)
google-generativeai>=0.8.0

# Option B: New SDK (Recommended)
google-genai>=0.3.0
```

**📖 Reference:** [Google GenAI Developer Guide](./GOOGLE_GENAI_DEVELOPER_GUIDE.md) | [Migration Guide](https://ai.google.dev/gemini-api/docs/migrate) | [Official Documentation](https://googleapis.github.io/python-genai/)
