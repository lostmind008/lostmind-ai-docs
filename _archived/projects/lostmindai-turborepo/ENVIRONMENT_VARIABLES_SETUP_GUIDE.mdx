## 1. Database Setup

### PostgreSQL with Row Level Security (RLS)

#### Option A: Supabase (Recommended for MVP)

**Dashboard**: https://supabase.com/dashboard

**Steps**:

1. Sign up/login at [Supabase Dashboard](https://supabase.com/dashboard)
2. Click **"New Project"**
3. Choose organization ‚Üí Enter project name: `lostmind-ai-dev`
4. Select region: **US East** (closest to Vercel)
5. Set database password (save this!)
6. Click **"Create new project"** (wait 2-3 minutes)

**Get Connection Strings**:

1. Go to **Settings** ‚Üí **Database**
2. Scroll to **Connection string**
3. Copy **"Postgres"** connection string
4. Replace `[YOUR-PASSWORD]` with your actual password

```bash
# Add to .env.local
DATABASE_URL="postgresql://postgres:[YOUR-PASSWORD]@db.[PROJECT-REF].supabase.co:5432/postgres"
DIRECT_URL="postgresql://postgres:[YOUR-PASSWORD]@db.[PROJECT-REF].supabase.co:5432/postgres"
```

#### Option B: Neon (Alternative)

**Dashboard**: https://console.neon.tech/app/projects

**Steps**:

1. Sign up at [Neon Console](https://console.neon.tech)
2. Click **"Create a project"**
3. Enter name: `lostmind-ai`
4. Select region: **US East (Virginia)**
5. Click **"Create project"**

**Get Connection String**:

1. In project dashboard, click **"Connection string"**
2. Copy the **"Pooled connection"** string

```bash
# Add to .env.local
DATABASE_URL="postgresql://[user]:[password]@[endpoint]/[dbname]?sslmode=require"
DIRECT_URL="postgresql://[user]:[password]@[endpoint]/[dbname]?sslmode=require&pgbouncer=true&connect_timeout=10"
```

**Verification**:

```bash
# Test connection
pnpm run db:push
# Should see "Your database is now in sync with your schema"
```

## 2. Authentication Setup

### Auth.js v5 + Google OAuth

#### NextAuth Secret Generation

```bash
# Generate a secure secret
openssl rand -base64 32

# Add to .env.local
NEXTAUTH_SECRET="[generated-32-char-string]"
NEXTAUTH_URL="http://localhost:3000"
```

#### Google OAuth Setup

**Dashboard**: https://console.cloud.google.com/apis/credentials

**Steps**:

1. Go to [Google Cloud Console](https://console.cloud.google.com)
2. Select your project: **coral-muse-469919-m0**
3. Navigate to **APIs & Services** ‚Üí **Credentials**
4. Click **"+ CREATE CREDENTIALS"** ‚Üí **"OAuth 2.0 Client IDs"**

**Configure OAuth Client**:

1. **Application type**: Web application
2. **Name**: `LostMind AI - Web Client`
3. **Authorized JavaScript origins**:
   ```
   http://localhost:3000
   http://localhost:3001
   http://localhost:3002
   https://lostmindai.com
   https://ask.lostmindai.com
   ```
4. **Authorized redirect URIs**:
   ```
   http://localhost:3000/api/auth/callback/google
   http://localhost:3001/api/auth/callback/google
   https://lostmindai.com/api/auth/callback/google
   https://ask.lostmindai.com/api/auth/callback/google
   ```
5. Click **"CREATE"**

**Get Credentials**:

1. Copy **Client ID** and **Client Secret** from the popup
2. Add to `.env.local`:

```bash
GOOGLE_CLIENT_ID="[client-id].apps.googleusercontent.com"
GOOGLE_CLIENT_SECRET="[client-secret]"
```

**Verification**:

1. Start dev server: `pnpm run dev`
2. Go to `http://localhost:3000/api/auth/signin`
3. Should see Google sign-in option
