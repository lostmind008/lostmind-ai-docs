### 2. Universal Processing
Process any type of file or URL with Gemini.

**Endpoint:** `POST /process`

**Request Body:**
```json
{
  "file_source": "string",  // Required: URL, file path, or base64 data
  "file_type": "string",    // Optional: auto-detected if not provided
  "prompt": "string",       // Required: instruction for processing
  "features": {             // Optional: advanced features
    "grounding": false,
    "code_execution": false,
    "youtube_chapters": true,
    "structured_output": null,
    "temperature": 0.7,
    "max_output_tokens": 8192,
    "response_mime_type": "text/plain"
  },
  "cache_enabled": true     // Optional: use caching (default: true)
}
```

**File Source Formats:**
- **URL**: `https://example.com/document.pdf`
- **YouTube**: `https://youtube.com/watch?v=VIDEO_ID`
- **Base64**: `data:application/pdf;base64,BASE64_CONTENT`
- **File Path**: `/path/to/file.txt` (for local development)

**Response:**
```json
{
  "success": true,
  "result": "Analysis results here...",
  "metadata": {
    "source_type": "url",
    "file_type": "document",
    "prompt_length": 45,
    "processing_time": 3.456,
    "features_used": {
      "grounding": false,
      "temperature": 0.7
    },
    "cache_hit": false
  },
  "cache_key": "abc123..."
}
```

### 3. YouTube Processing
Specialized endpoint for YouTube video analysis.

**Endpoint:** `POST /youtube`

**Request Body:**
```json
{
  "url": "https://youtube.com/watch?v=VIDEO_ID",
  "prompt": "Summarize the key points",
  "features": {
    "temperature": 0.7,
    "max_output_tokens": 8192
  },
  "extract_captions": true,
  "time_range": {
    "start": 0,
    "end": 300  // seconds
  }
}
```

**Response:**
```json
{
  "success": true,
  "video_id": "VIDEO_ID",
  "analysis": "Video analysis results...",
  "metadata": {
    "model": "gemini-2.5-flash",
    "features_used": {...}
  }
}
```

### 4. Code Analysis
Analyze and improve code snippets.

**Endpoint:** `POST /code`

**Request Body:**
```json
{
  "code": "def hello():\n    print('world')",
  "language": "python",
  "prompt": "Optimize this function",
  "test_inputs": [1, 2, 3],
  "execute": true
}
```

**Supported Languages:**
- Python, JavaScript, Java, C++, C, Go, Rust, PHP, Ruby, Swift, Kotlin, Scala, SQL, YAML

**Response:**
```json
{
  "success": true,
  "language": "python",
  "original_code": "...",
  "analysis": "Code analysis and improvements...",
  "improvements": "See analysis above"
}
```

### 5. Research Assistant
Research topics with comprehensive analysis.

**Endpoint:** `POST /research`

**Form Data:**
- `topic` (required): Topic to research
- `depth` (optional): "quick", "moderate", "comprehensive" (default)
- `include_citations` (optional): true/false (default: true)

**Response:**
```json
{
  "topic": "quantum computing",
  "depth": "comprehensive",
  "research": "Detailed research report...",
  "model": "gemini-2.5-flash"
}
```
